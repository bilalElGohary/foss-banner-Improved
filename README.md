## ūüöÄ Getting Started

### Requirements

- Python 3.x
- A terminal that supports ANSI escape codes (most modern terminals do)

### Installation

```bash
git clone https://github.com/detectiveRight/foss-banner.git
cd foss-banner
```

### Usage

```bash
python foss-banner.py
```
